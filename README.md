# Railway
A full-stack train search web application built using Node.js, Express, and MySQL, allowing users to find trains between cities with real-time results using optimized SQL queries.

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express  
- Database: MySQL (Railway Cloud)  

##  Features
- Search trains by source and destination  
- View departure, arrival, and duration  
- Case-insensitive and flexible search  

##  Deployment
- Backend: Railway  
- Frontend: Railway
